# n8n
Cài n8n với Node.js 22 qua nvm, PostgreSQL, PM2, Nginx, SSL Let’s Encrypt
# Cách cài đặt:
Các bạn login SSH và chạy các lệnh sau:
<pre>
wget https://raw.githubusercontent.com/bnixvn/n8n/refs/heads/main/n8n.sh
chmod +x n8n.sh
./n8n.sh
</pre>
